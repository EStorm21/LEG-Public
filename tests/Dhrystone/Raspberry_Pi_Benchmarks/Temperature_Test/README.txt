See RPiTemperature.c for information on how to compile and run
and  RPiTemperature.txt for results log file. The program is 
run via a command in an LXTerminal window.

                    for ppp samples at sss second intervals
                    only needs first character p, P, s, S
 15 minutes test -  ./RPiTemperature passes 30, seconds 30

 Default            ./RPiTemperature
